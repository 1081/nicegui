# Nice GUI

## Usage

Write your nice GUI in a file `main.py`:

    from nice_gui import ui

    ui.label('Hello Nice GUI!')
    ui.button('BUTTON', on_click: lambda: print('button was pressed'))

Launch it with:

    python3 main.py
